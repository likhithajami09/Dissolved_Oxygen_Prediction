# Dissolved Oxygen Prediction
This project predicts dissolved oxygen levels in water using machine learning techniques.

## Features
- ML-based prediction model
- Flask web application
- Dataset processing
- Visualization support

## Tech Stack
- Python
- Flask
- LightGBM
- Pandas
- Scikit-learn

## How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python app.py

3. Open browser:
http://127.0.0.1:5000/

## Project Structure
- Dataset/
- model/
- templates/
- static/
- app.py
